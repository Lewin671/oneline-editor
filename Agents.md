# Technology Stack

## Frontend (Web)

- **Framework**: Next.js 16 (App Router, Server Actions)
- **Language**: TypeScript
- **UI & Styling**: shadcn/ui, Tailwind CSS, Lucide React and Vercel AI SDK for prebuilt AI elements
- **State Management**: Zustand (Client-side), React Server Components (Server-side)
- **Validation**: Zod
- **Infrastructure**: Vercel
- **Lint/Format**: Biome
- **Package Management**: pnpm

## Backend (Server)

- **Runtime**: Node.js
- **Language**: TypeScript
- **Framework**: Express
- **WebSocket**: ws
- **LSP Protocol**: vscode-languageserver-protocol, vscode-jsonrpc
- **Testing**: Vitest, fast-check (Property-based testing)
- **Development**: tsx (TypeScript execution)
- **Build**: TypeScript Compiler (tsc)

## Core Features

- **Language Server Protocol (LSP)**: Native support for code intelligence features
- **Virtual File System**: In-memory file management
- **Monaco Editor**: Code editor integration
- **Real-time Communication**: WebSocket-based LSP message transport
